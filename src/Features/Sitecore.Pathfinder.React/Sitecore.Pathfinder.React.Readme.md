You must install React.NET manually on the website.

Copy these assemblies to /bin:

* ClearScript.dll
* JavaScriptEngineSwitcher.Core.dll
* JavaScriptEngineSwitcher.Msie.dll
* JavaScriptEngineSwitcher.V8.dll
* JSPool.dll
* MsieJavaScriptEngine.dll
* React.Core.dll
* React.Web.dll
* React.Web.Mvc4.dll
* VroomJs.dll
* WebActivatorEx.dll

init-gulp
=========
Configures the project to use GulpJS.

Remarks
-------
The 'init-gulp' task adds a 'gulpfile.js' to the root of the project directory.

The task simply unzips the [Tools]/files/taskrunners/Gulp.zip file into the project.

Settings
--------
None.

Example
-------
```cmd
> scc init-gulp
```

list-items
===============
Lists the project items (Sitecore items and files)

Remarks
-------
The `list-project` tasks outputs a list of all the items and files in the project (including imported items).

Settings
--------
None.

Example
-------
```cmd
> scc list-project

/sitecore (Item)
/sitecore/content (Item)
/sitecore/layout (Item)
```

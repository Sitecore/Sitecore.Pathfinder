# Json layout format
To specify a layout in Json, use the format below.

```js
{
    "Item": {
        "Layout": {
            "Devices": [
                {
                    "Name": "Default",
                    "Layout": "/sitecore/layout/Layouts/MvcLayout",
                    "Renderings": [
                        {
                            "HelloWorld": { "Text": "Welcome" }
                        },
                        {
                            "BodyText": { }
                        },
                        {
                            "Footer": { }
                        }
                    ]
                }
            ]
        }
    }
}
```
